<h1>📊 Painel de Benefícios Fiscais - ICMS/SP</h1>
    <p>Uma solução técnica e visual desenvolvida para simplificar a consulta e gestão de benefícios fiscais no Estado de
        São Paulo (<strong>RICMS/SP</strong>). Este projeto transforma dados complexos de <strong>CST</strong> (Código
        de Situação Tributária) e <strong>CBenef</strong> (Código de Benefício) em um relatório intuitivo, facilitando a
        vida de contadores, faturistas e gestores.</p>
    <h2>🚀 Principais Funcionalidades</h2>
    <h3>🔍 Busca e Filtragem Inteligente</h3>
    <ul>
        <li>
            <p><strong>Filtro por CST:</strong> Visualize rapidamente itens com Redução de Base (20), Isenção com ST
                (30) ou Isenção Total (40).</p>
        </li>
        <li>
            <p><strong>Busca Global:</strong> Encontre itens por nome do produto, código CBenef ou base legal em tempo
                real.</p>
        </li>
    </ul>
    <h3>📄 Relatório Técnico Exportável (PDF A4)</h3>
    <ul>
        <li>
            <p><strong>Formatação Profissional:</strong> Gerador de PDF otimizado para papel A4.</p>
        </li>
        <li>
            <p><strong>Controle de Quebra de Página:</strong> Algoritmo que impede que blocos de informações sejam
                cortados entre páginas, garantindo um documento limpo.</p>
        </li>
        <li>
            <p><strong>Modo Impressão:</strong> Ajuste automático de fontes e cores para economia de tinta e melhor
                legibilidade no papel.</p>
        </li>
    </ul>
    <h3>💡 Visualização Estruturada</h3>
    <ul>
        <li>
            <p><strong>Cards Coloridos:</strong> Identificação visual imediata do tipo de tributação através de cores
                semânticas.</p>
        </li>
        <li>
            <p><strong>Regras de Redução:</strong> Exibição clara das alíquotas de carga tributária final (ex: 7% ou
                12%) e prazos de vigência.</p>
        </li>
        <li>
            <p><strong>Destaque de Códigos:</strong> Foco total nos códigos operacionais (<strong>CST</strong> e
                <strong>CBenef</strong>) para evitar erros de digitação no ERP.</p>
        </li>
    </ul>
    <h2>🛠️ Tecnologias Utilizadas</h2>
    <ul>
        <li>
            <p><strong>Frontend:</strong> HTML5 e CSS3 (Design Moderno).</p>
        </li>
        <li>
            <p><strong>Estilização:</strong> <a href="https://tailwindcss.com/" title="null">Tailwind CSS</a> (Layout
                responsivo e utilitários).</p>
        </li>
        <li>
            <p><strong>Ícones:</strong> <a href="https://lucide.dev/" title="null">Lucide React</a> (Ícones vetoriais
                leves).</p>
        </li>
        <li>
            <p><strong>Motor de PDF:</strong> <a href="https://ekoopmans.github.io/html2pdf.js/"

        </li>
        <li>
            <p><strong>Tipografia:</strong> Inter (Google Fonts).</p>
        </li>
    </ul>
    <h2>📦 Como Executar o Projeto</h2>
    <p>O projeto é <strong>estático (Frontend-only)</strong>, o que significa que não precisa de servidor ou banco de
        dados para funcionar.</p>
    <ol>
        <li>
            <p>Faça o download do arquivo <code>dashboard.html</code>.</p>
        </li>
        <li>
            <p>Abra o arquivo diretamente em seu navegador (Chrome, Firefox, Edge ou Safari).</p>
        </li>
        <li>
            <p>Pronto! Você já pode consultar os dados e gerar relatórios.</p>
        </li>
    </ol>
